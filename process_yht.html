Overview

보다 복잡한 구조의 디자인을 구현해보고자 작업한 여행사 사이트로, 레이아웃의 구성과 효과적인 구현에 집중 하였습니다.
평소 자주 접하는 유형의 사이트가 아니기 때문에 비슷한 분야의 웹사이트 디자인들을 벤치마킹하여 컨셉에 따라 리디자인 하였습니다.
반응형으로 제작하여 PC와 모바일 모두 이용 가능하도록 하였습니다.

Project Participation

  • Planning
    2020.10.15 - 10.16

    비슷한 분야의 여러 사이트들을 벤치마킹 하며 '여행사' 사이트의 공통된 구조와 콘텐츠들을 파악하고, 사이트맵과 콘텐츠 구성을 계획하였습니다. '여행'이라는 타이틀에 걸맞는 컨셉을 잡고, 다양한 레이아웃과 디자인들을 수집 하였습니다.

  • Design
    2020.10.19 - 10.20

    브랜드를 이미지화 하고 디자인에 반영 하였습니다. '여행'이라는 키워드에서 느껴지는 시원함과, 신뢰감 있는 이미지를 딥한 색상으로 나타내어 포인트 컬러를 지정 하였습니다. 복잡하고 다양한 레이아웃을 운율감 있게 배치 하였습니다.

  • Modification
    2020.10.20

    전체적으로 통일감 있도록 디자인을 다듬고 클릭시에 구현할 요소들의 디자인을 추가 작업 하였습니다.

  • Html/CSS
    2020.10.21 - 10.24

    복잡한 레이아웃을 효과적으로 구조화 하였고, 추후 수정 및 보완의 용이하며, 모바일에서도 사용 가능하도록 설계 하였습니다.

  • jQuery
    2020.11.04 - 11.11

    슬라이드 배너와 실제 버튼 클릭시에 구현되는 효과 등의 디테일 요소들을 추가 하였습니다. 30%는 외부 플러그인을 수정 및 보완하여 사용한 부분이 있고, 당시 전문 강사님께 도움을 받았습니다. 현재는 완벽히 이해하여 응용이 가능합니다.

Epilogue

더 빠르게 더 편리하게, 하나라도 제대로.

'복잡한 사이트의 메인 하나라도 제대로 만들어보면 어떨까?'
코딩에 재미를 붙이면서 더 복잡한 레이아웃과 기능에 도전해보고 싶었습니다.
여러 여행사 사이트들을 비교 분석해보며 더 효율적인 이용이 가능한 설계에 대해 고안하였습니다.
끊임없이 질문하고 찾아보고 수정하여 '반응형'과 '웹 폼', '달력' 등의 기능이 추가 된 사이트를 완성 했습니다.
이 과정에서 학원에서 배우지 못했던 여러 코드들을 스스로 찾아 분석·습득하여 적용 하였습니다.
자바스크립트에 입문하여 같이 공부했던 학생들이 코딩을 포기하고 시각 디자인으로 돌아설 때,
저는 오히려 머리 속 동적 이미지를 웹 상에 구현해냈을 때의 성취감에 가슴이 뛰었습니다.
코딩에 대한 흥미와 이해도를 한 단계 더 업그레이드 할 수 있는 시간이었습니다.
